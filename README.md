### Requirements

Java 17+

### Run tests

#### macOS
```bash
./gradlew build
```
#### Windows
```bash
.\gradlew.bat build
```

### Create a visual Allure Report for executed tests
#### macOS
```bash
./gradlew allureServe
```
#### Windows
```bash
.\gradlew.bat allureServe
```
